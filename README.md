# Idea to Code Platform 

A collaborative web platform built using Django that allows developers to:

- Create project ideas
- Send join requests
- Approve contributors
- Manage team members
- View dashboard analytics
- Maintain user profiles

## Tech Stack

- Django
- Bootstrap
- SQLite
- HTML/CSS
- Git

## Features

- User Authentication (Signup/Login)
- Profile Extension using OneToOneField
- Project Creation
- Join Request Approval Workflow
- ManyToMany Team System
- Dashboard with Analytics
- Role-based UI Rendering

## How to Run

1. Clone repository
2. Create virtual environment
3. Install dependencies
4. Run migrations
5. Run server
